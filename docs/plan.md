# VoiceInput 実装計画

## 概要

Windows 向け SuperWhisper 風ローカル音声入力アプリ。

### 主要機能

1. グローバルショートカットで任意アプリ上から音声入力
2. 日本語音声認識（Whisper large-v3-turbo）
3. LLM による文脈理解・整形（カスタマイズ可能）
4. 完全オフライン処理
5. 音声入力ログ保存
6. モデル別途ダウンロード
